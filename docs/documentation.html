全球温室气体排放趋势探索器｜说明文档

目标：用交互式探索回答“自1990年以来，不同国家的排放如何变化？谁在显著减排，谁在持续增长？”并支持用户发现拐点、比较政策前后差异。

1. 可视化目标（回答的问题）

  • 主问题：自1990年以来，各国排放（你的数据字段 value）随时间如何变化？
  • 对比：用户可自由选择多个国家对比趋势形状与阶段性变化。
  • 洞察：在任意时间区间内，快速识别“显著减排/显著增排”的国家与变化幅度。
  • 异常/拐点：通过缩放、刷选与指数化视图更容易发现波动与转折。

2. 设计决策（为什么这样设计）

为什么是折线图

折线图最适合表达长期趋势、拐点与阶段性变化；支持同时比较多条国家曲线。

为避免缺失年份造成“虚假直连”，系统会在年份缺失处断线显示。

为什么要“指数化（区间起点=100）”

当国家量级差异大时，原始值会掩盖小国趋势；指数化把焦点转向“增长/下降速度与形状”。

基准取刷选区间内该国的首个有效值，确保与用户当前探索区间一致。

为什么要刷选 + 缩放（双交互）

刷选用于快速锁定政策前后区间；缩放/平移用于细读局部变化，避免反复拖拽刷选。

为什么要“区间变化排行”视图

它直接回答“谁在显著减排/增排”，并且与主图联动：刷选区间一变，排行立刻更新。

3. 多视图协调（联动机制）

  • 刷选 ↔ 主图：底部缩略图刷选时间区间，主图自动更新尺度与内容。
  • 缩放 ↔ 刷选：在主图滚轮缩放/拖拽平移，会同步更新底部刷选窗口（避免两套时间范围“打架”）。
  • 主图悬停 ↔ 排行高亮：主图悬停某一年，提示框展示多国对齐值；右侧排行同步高亮这些国家，便于上下文判断。
  • 排行点击 ↔ 国家选择：点击右侧条形图即可增删国家，实现“从洞察反推对比对象”的探索路径。

4. 高级交互与创新元素

  • 高级交互：缩放/平移（zoom）、时间刷选（brush）、悬停对齐（crosshair + 多国 tooltip）、图例隐藏/显示。
  • 创新元素：区间变化的发散条形排行（以 0 为中心），并支持点击反向选择；“指数化模式”把探索从“量级”切换到“趋势形状”。
  • 政策事件时间轴联动：预置政策/宏观事件；主图琥珀色虚线竖线 + 可点击热区与标签（支持 Enter/空格），点击后以事件年为中心按 padYears 扩展刷选区间并与数据年份范围求交；缩略图底部小三角与开关联动。
  • 异常点与解释面板：基于各国相邻年排放差分的均值±2σ 标出“异常波动”，并补充 |Δ| 最大的 Top 2 年为“显著变化”；主图空心圆标记，悬停/点击后右侧面板展示差值、全球排名变化与阈值带说明。

5. 数据源与预处理说明

本项目数据源来自于Greenhouse Gas Emissions, 1990-2019经济合作与发展组织（OECD），经预处理综合为 emissions.json，数据结构为：

[{ "country": "...", "year": 1990, "value": 12.34 }, ...]

为保证可视化不误导，前端会：

  • 将每个国家扩展为“全年份序列”，缺失年份填 null 以产生断线。
  • 排行计算基于刷选区间内该国首末有效年份（可应对国家起始年份不同、年份稀疏）。

6. 参考与启发

  • D3 Focus + Context 经典模式：主视图 + 缩略图刷选。
  • NameVoyager：通过交互筛选与即时反馈形成探索节奏。
  • Zipdecode：强调“从概览到细节”的多层级导航。

7. 开发流程与难点

  • 难点：zoom 与 brush 的同步（避免相互触发造成抖动/循环）。
  • 难点：缺失年份的正确处理（断线显示 + 计算首末有效值）。
  • 收获：把“洞察视图（排行）”做成可交互入口，比单纯的折线图更能引导探索。

8. 未来改进

  • 增加“政策事件库”（可勾选多个垂直标注，支持自定义事件）。
  • 增加“拐点检测”（例如滑动窗口斜率变化提示）。
  • 增加人均/总量切换（若补充人口数据），形成更完整的指标视角。
